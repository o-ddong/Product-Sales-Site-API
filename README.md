### git clone
```
git clone https://github.com/o-ddong/fruitte.git
```
### 가상환경 생성
```
python -m venv venv
```

### 가상환경 접속
```
source /venv/bin/activate
```

### requirements 설치
```
pip install -r requestments
```

### 서버 실행
```
python manage.py runserver
```
