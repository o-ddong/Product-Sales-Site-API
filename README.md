### git clone
```
git clone https://github.com/o-ddong/fruitte.git
```
### 가상환경 생성
```
python -m venv venv
```

### 가상환경 접속
```
source /venv/bin/activate
```

### requirements 설치
requirements 파일 경로가 있는 곳에서 다음과 같이 실행해주면 된다.
```
pip install -r requestments
```
